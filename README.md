# Polymarket Bot (Initial Coding Skeleton)

Bu repo, Polymarket botu için ilk kodlama adımını içerir:

- Geoblock fail-safe guard
- Basit sequence reconciliation yardımcıları
- İlk risk limiti motoru (order gate)
- Konfig doğrulama modeli

## Kurulum

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e . pytest
```

## Test

```bash
pytest
```
